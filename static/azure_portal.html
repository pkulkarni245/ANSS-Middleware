Configure Probabilistic Constraints for Azure AI

Use this portal to configure the Deterministic Guardrails (Phase 3). The Azure Fabric will compile these rules into `.prism` format and deploy them to the isolated ANSS Function App (`anss-dtmc-validator-func`).

How it works

Instead of traditional RBAC, ANSS mathematically models AI execution as a Markov Chain. Defining a constraint guarantees zero probability of an AI agent reaching an illegal state, regardless of prompt injection attacks.
